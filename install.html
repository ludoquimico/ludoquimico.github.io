Configurações de Monitor e Hardware
O jogo não necessita nenhuma configuração de hardware ou software específica. Ele foi projetado para funcionar em computadores antigos e com pouca memória, configuração geralmente encontrada nos laboratórios de Informática das escolas públicas (quando eles existem!).

No entanto, para ter uma melhor experiência de jogabilidade, aconselhamos utilizar o monitor com a configuração mínima de resolução 1366x768. Essa resolução de vídeo é comum em computadores de mesa e portáteis (como netbook e notebooks). Caso o monitor esteja configurado configurado para uma resolução menor do que a indicada, corre-se o risco de não se visualizar por completo as questões apresentadas na tela. Questões com muito texto, teriam que ser apresentados com uma fonte menor, algo que poderia ser prejudicial às pessoas com baixa visão.

A conexão com a Internet é necessária apenas para a atualização do software ou da base de questões. Caso não seja possível atualizar o jogo pela Internet, peça para alguém com conexão que faça o download dos arquivos dos links LINK 1 e LINK 2 e os copie para a pasta de instalação do jogo.

Instalação Windows 7,8 e 10

 1) Faça o donwload do arquivo de instalação;
2) Execute o arquivo e siga os passos indicados;
3) Prefira instalar na pasta "Meus Documentos" ou outro local que não necessite permissões especiais de leitura e/ou gravação.

Para atualizar a base de questões do jogo, clique como o botão direito do mouse no ícone "Atualizar base de questões" no Desktop ou no Menu Iniciar, conforme imagens abaixo.





Clique em "Executar como administrador" e aguarde a mensagem de atualização do programa.

Instalação Linux Debian e Raspbian

1) Faça o donwload do arquivo de instalação (arquivo.deb);
2) Abra o terminal de sua preferência;
3) Em modo ROOT, instale as seguintes dependências, conforme comando abaixo:

usuário$ sudo apt-get install libphysfs1 liballegro5.2 liballegro-acodec5.2 liballegro-audio5.2 liballegro-image5.2 liballegro-physfs5.2 liballegro-ttf5.2 liballegro-video5.2 libsqlcipher0 libcurl4
4) Ainda em modo ROOT, instale o (arquivo.deb) copiado do site, conforme comando abaixo:

usuário$ dpkg -i arquivo.deb

5) No terminal, em modo usuário, crie uma pasta de sua preferência;
6) Entre na pasta criada e execute o comando:

usuário$ ludo_quimico

O primeiro uso faz o download da base de questões para a pasta que acabou de criar. Se estiver sem Internet, você pode copiar manualmente os arquivos ludo_quimico.dat e ludo_quimico.zip que estão na pasta /usr/bin para a pasta que acabou de criar. Certifique-se que o seu usuário tem permissão para realizar essas ações.

Para novas atualizações na base de questões, pelo terminal do Linux entre novamente na pasta que você criou e execute o seguinte comando:

usuário$ ludo_quimico --update

Esse comando faz o donwload da última base de questões atualizada.
